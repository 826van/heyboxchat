# 关键词采集 Edge 插件合集

这个文件夹下有两个可直接加载到 Edge 的扩展目录：

- `heybox`：小黑盒关键词采集插件。
- `xiaohongshu`：小红书关键词采集插件。

## 安装到 Edge

1. 打开你平时使用的 Edge。
2. 在地址栏输入 `edge://extensions/`，按回车。
3. 打开「开发人员模式」。
4. 点击「加载解压缩的扩展」。
5. 在文件夹选择窗口里选择具体插件目录：
   - 小黑盒：`E:\edge插件\heybox`
   - 小红书：`E:\edge插件\xiaohongshu`
6. 点击「选择文件夹」完成加载。

不要选择 `E:\edge插件` 这个总目录，要选择里面的 `heybox` 或 `xiaohongshu` 子目录。两个插件都要装，就重复点击「加载解压缩的扩展」并分别选择两个子目录。

加载成功后，`edge://extensions/` 页面会看到：

- `Xiaoheihe Keyword Helper`
- `Xiaohongshu Keyword Helper`

如果常用 Edge 里没有看到插件，通常是因为插件装到了另一个 Edge 配置文件或测试窗口里。回到你平时使用的 Edge 窗口，再按上面步骤加载一次。

## 更新后重新加载

代码更新后，打开 `edge://extensions/`，找到对应插件卡片，点击「重新加载」，然后刷新网站页面。如果插件目录移动过，先移除旧插件，再重新加载新目录。

两个插件都只做关键词扫描、草稿复制和打开预填；评论发布仍然需要你自己手动检查并点击网站原生发布按钮。

小红书插件的「打开并预填」会自动展开评论框并写入随机草稿。为了让真实页面接受自动填写，它需要 Edge 的 `debugger` 权限；重新加载插件时如果 Edge 提示确认权限，选择允许即可。插件只用于点击评论框和写入文字，不会自动点击发布。
