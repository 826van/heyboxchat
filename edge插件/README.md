# Edge 插件集合

这个文件夹下有两个可直接加载到 Edge 的扩展目录：

- `heybox`：小黑盒关键词采集插件。
- `xiaohongshu`：小红书关键词采集插件。

安装时打开 `edge://extensions/`，开启「开发人员模式」，然后分别选择对应子目录加载：

- `E:\edge插件\heybox`
- `E:\edge插件\xiaohongshu`

两个插件都只做关键词扫描、草稿复制和打开预填；评论发布仍然需要你自己手动检查并点击网站原生发布按钮。

小红书插件的「打开并预填」会自动展开评论框并写入随机草稿。为了让真实页面接受自动填写，它需要 Edge 的 `debugger` 权限；重新加载插件时如果 Edge 提示确认权限，选择允许即可。插件只用于点击评论框和写入文字，不会自动点击发布。
